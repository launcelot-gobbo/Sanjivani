npm install express mongoose ejs

1. node app.js 
2. check if mongoDB compass is running (mongodb://127.0.0.1:27017/music)
3. Open browser at localhost:3000

Insert songs: http://localhost:3000/insert (run only once)
View songs: http://localhost:3000/
Director filter: http://localhost:3000/director/Pritam
Director + Singer: http://localhost:3000/director/Pritam/singer/Arijit%20Singh
Delete: http://localhost:3000/delete/Jai%20Ho
Update actor/actress: http://localhost:3000/update/Tum%20Hi%20Ho/Shahrukh%20Khan/Katrina%20Kaif

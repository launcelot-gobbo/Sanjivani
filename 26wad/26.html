PICT's B.E.course structure.

First Year[FE]

Mathematics I

Introduction to mathematical concepts for engineering students.

Physics I

Fundamentals of physics applied to engineering problems.

Programming Basics

Introduction to computer programming and problem-solving.

Engineering Mechanics

Basic principles of mechanics and their applications in engineering.

Second Year[SE]

Mathematics II

Advanced mathematical techniques for engineers.

Digital Logic Design

Study of digital circuits and logic gates.

Object-Oriented Programming

Introduction to OOP concepts using C++ or Java.

Data Structures

Fundamentals of data structures and algorithms.

Third year[TE]

Operating Systems

Study of system software and the operating systems.

Database Management Systems

Introduction to database systems, design, and queries.

Computer Networks

Study of data communication and computer networking concepts.

Software Engineering

Principles and practices of software development and maintenance.

Final Year[BE]

Advanced Data Structures

Advanced topics in data structures and their applications.

Machine Learning

Introduction to machine learning algorithms and models.

Web Technologies

Development of modern web applications using various technologies.

Project Work

Practical project implementation and documentation.
